# Методы обработки изображений

Репозиторий организован как набор отдельных проектов по лабораторным работам.

## Структура

- `labs/lab_01` — ЛР 1. Вычисление яркости на поверхности треугольника
- `labs/lab_02` — заготовка под ЛР 2
- `labs/lab_03` — заготовка под ЛР 3

Каждая лабораторная живет в своей папке как отдельный проект со своей структурой:

```text
lab_xx/
  src/
  tests/
  examples/
  report/
  README.md
```

## Текущая готовность

- ЛР 1 реализована полностью, включая GUI, CLI, рендер из точки наблюдателя, тесты и Typst-отчет
- ЛР 2 и ЛР 3 подготовлены как отдельные самостоятельные каталоги для дальнейшей разработки
