# Методы обработки изображений

Репозиторий организован как набор отдельных проектов по лабораторным работам.

## Структура

- `labs/lab_01` — ЛР 1. Вычисление яркости на поверхности треугольника
- `labs/lab_02` — ЛР 2. Monte Carlo Variance Studio
- `labs/lab_03` — ЛР 3. Формирование распределений случайных величин
- `labs/lab_04` — ЛР 4. Формирование изображения методом трассировки путей

Каждая лабораторная живет в своей папке как отдельный проект со своей структурой:

```text
lab_xx/
  src/
  tests/
  examples/
  report/
  README.md
```

## Текущая готовность

- ЛР 1 реализована полностью, включая GUI, CLI, рендер из точки наблюдателя, тесты и Typst-отчет
- ЛР 2 и ЛР 3 реализованы как отдельные самостоятельные проекты с GUI, CLI, тестами и Typst-отчетами
- ЛР 4 содержит path tracer для треугольных сеток, GUI/CLI, demo-сцену, тесты и Typst-отчет
